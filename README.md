# masnmapscan-GO

这是https://github.com/hellogoldsnakeman/masnmapscan-V1.0 的GO版本

Masscan + Nmap 的工具GO版本

MASSCAN 在本地编译 放在与程序相同目录

nmap 直接使用apt install nmap 进行安装即可



我的测试环境是Ubnatu 22 此脚本接入我自己的渗透测试自动化流程中 

流程走势如下

正在使用subfinder扫描子域名...
正在使用ksubdomain验证子域名...
正在使用httpx进行HTTP探测...
正在使用masscan和nmap进行端口扫描...
